# FORMULARIOS 
## no HTML
 para escrever dentro de um input usamos:  
```` html
<input type="text" placeholder="escreve dentro do input" > 
```` 
 para escrever em um formulario usamos:  
```` html
<label for="o id do input" > 
````
 para desabilitar um input usamos:   
````html
<input disabled="" >
````
